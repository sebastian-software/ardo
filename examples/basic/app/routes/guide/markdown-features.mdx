# Markdown Features

Ardo supports standard Markdown, GitHub Flavored Markdown (GFM), and several extensions.

## Code Blocks

Syntax highlighting is powered by Shiki:

```typescript
interface User {
  id: string
  name: string
  email: string
}

function greetUser(user: User): string {
  return `Hello, ${user.name}!`
}
```
